# eslint-config-redduck

ESLint config for plain TS which RedDuck Software uses in its projects.

## How to install

- npm

  ```shell
  npm install eslint-config-redduck -D
  ```

- yarn

  ```shell
  yarn add eslint-config-redduck -D
  ```

- pnpm

  ```shell
  pnpm add eslint-config-redduck -D
  ```
